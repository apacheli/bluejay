# What's New in Bluejay 0.2.1?

## Install

```
$ bun add https://github.com/apacheli/bluejay
```

## QoL Changes

- You can now use `bluejay/preload` in `bunfig.toml` for preloading modules.
- Added TypeScript documentation for `BluejayOptions`.

## Bug Fixes

- Fixed package exports.
