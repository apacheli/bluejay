# What's New in Bluejay 0.4.1?

## Install

```
$ bun add https://github.com/apacheli/bluejay
```

## Bug Fixes

- Fixed imports for `.md` to use Markdown instead of MDX. `.markdown` files can also now be parsed.
- Preload plugins are no longer case sensitive.
